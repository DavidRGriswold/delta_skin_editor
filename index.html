Instructions

  1. Set your name, id, and game type at the top.
  2. Flip between the various canvases and set up your buttons by dragging and dropping.
  3. Use the "Download PDF" button one at a time to download all four PDFs.
  4. Use the "Download JSON" button to download your info.json file
  5. On your computer, make a ZIP file that includes all five files.
  6. Rename the file with the extension ".deltaskin"
  7. Put on phone. Profit??

Limitations

  1. Right now, the checkboxes don't do anything -- all four screens will be included always
  2. I don't think the .deltaskin file can have an internal folder - the files need to be at the top level. That can be tricky with built-in zip tools, try 7-zip.
  3. Resizing doesn't work yet.
